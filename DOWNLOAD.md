# Download DjVu to PDF Converter

## For Windows Users

### Easy Download (Recommended)

1. Go to the [**Actions**](../../actions) tab in GitHub
2. Click on the latest successful build (green checkmark ✓)
3. Scroll down to **"Artifacts"**
4. Download **`djvu2pdf-windows`**
5. Extract the ZIP file
6. Run **`djvu2pdf.exe`**

That's it! No installation needed!

### Alternative: Download from Releases

If a release has been published:

1. Go to [**Releases**](../../releases)
2. Download the latest **`djvu2pdf.exe`**
3. Run it!

## How to Use

1. **Double-click** `djvu2pdf.exe`
2. **Drag your DjVu file** into the window (or click "Select File")
3. **Click "Convert to PDF"**
4. **Choose where to save** your PDF
5. **Wait** for conversion (progress bar shows status)
6. **Done!** Your searchable PDF is ready

## Troubleshooting

### "Windows protected your PC"
- Click "More info" → "Run anyway"
- Or: Right-click the .exe → Properties → Check "Unblock" → OK

### "Cannot open file"
- Make sure you're selecting a `.djvu` or `.djv` file
- Try a different DjVu file to test

### "Conversion failed"
- Check that the DjVu file isn't corrupted
- Ensure you have enough disk space
- Make sure the output folder isn't read-only

### Antivirus Blocking
- Some antivirus software may flag unknown executables
- Add an exception for `djvu2pdf.exe`
- The software is open-source and safe
